# expense-tracker
Backend ứng dụng theo dõi chi tiêu cá nhân

- Install swag
```shell
# https://github.com/swaggo/echo-swagger
go install github.com/swaggo/swag/cmd/swag@latest
```

- Run server admin
```shell
make run-admin
```

- Swagger
```shell
Local: http://localhost:8000/admin/expense/swagger/index.html
```
